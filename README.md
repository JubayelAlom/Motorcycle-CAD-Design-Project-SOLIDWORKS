# Motorcycle CAD Design Project – SOLIDWORKS

## Overview
This project is a complete motorcycle CAD design and assembly developed in SOLIDWORKS as part of the Udemy course:

**“Mastering SOLIDWORKS (2024–25): A Complete Course”**

The project was created to improve my practical understanding of mechanical design, assembly workflow, weldment structures, and engineering drawing preparation using SOLIDWORKS.

![Video Alt](https://github.com/JubayelAlom/Motorcycle-CAD-Design-Project-SOLIDWORKS/raw/refs/heads/main/bike%20full%20design.mp4)
---

# Project Highlights
- Tubular Weldment Chassis Design
- Full Motorcycle Assembly
- Front & Rear Suspension Integration
- Wheel and Fork Assembly
- Engineering Drawing Creation
- Exploded View & BOM Preparation
- Mechanical Component Integration

---

# Components Included
- Weldment Chassis / Frame
- Fuel Tank
- Front & Rear Mudguards
- Seat & Seat Frame
- Shock Absorber Assembly
- Front Fork Assembly
- Wheels & Tires
- Handlebar
- Engine Components
- Headlight Components

# Learning Outcome
Through this project, I learned:
- How multiple mechanical components interact in a complete assembly
- Practical workflow of motorcycle CAD design
- Importance of frame geometry and structural member placement
- Basics of engineering documentation and BOM creation
- Real-world assembly visualization and detailing

This project helped strengthen both my CAD skills and my understanding of automotive/mechanical design concepts.
